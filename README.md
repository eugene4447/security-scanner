## How to Verify On-chain Verdict
Once the scan is complete, the AI report is saved to the Base blockchain. You can view the full verdict:
1. Copy the **Transaction Hash (TXID)** provided after the scan.
2. Go to [BaseScan](https://basescan.org/).
3. Paste the hash into the search bar.
4. Click **"More details"** to expand transaction information.
5. In the "Input Data" section, change the view to **"UTF-8"** to read the AI report directly on-chain.
